Layout
Это HOC. Используется для шеринга UI компонент.
Файл layout.tsx, расположенный в app/ui - root layout - разшарит свои UI компоненты между всеми страницами приложения.
Файл layout.tsx, расположенный в app/ui/dashboard - layout только для dashboard и вложенных страниц - разшарит свои UI компоненты между страницей dashboard и вложенными в папку dashboard

Link
Компонент.
В отличии от тега <a> не приводит к полной перезагрузке страницы.
Обеспечивает код склиттинг на основе путей - делает страницы изолированными друг от друга.
Обеспечивает префетчинг на основе путей - всякий раз, когда тег появляется в области видимости то просиходит фоновая загрузка кода, уда ведет ссылка, появившегося тега.
